유투브 동영상 반응형 제작 

1. <video> 요소로 동영상을 넣을수 있지만, 용량문제로 유투브에 올려서 링크하는 방식사용

2. 유투브 사이트 해당 동영상 재생상태에서 오른쪽 버튼 > 소스코드 복사하기 

3. html 문서에서 동영상 파일이 들어갈 영역을 지정한후 그안에 복사된 소스코드 붙여넣기 

4. 웹접근성 에서는 <iframe> 요소를 사용하는 것보다 <embed> 요소로 바꾸는 것이 좋음 

5. 동영상 관련 여러가지 옵션 설정 ( &로 계속 이어서 지정 ) 

      자동재생 :  ?autoplay=1&mute=1 ( 웹접근성 )     
      영상시작시점 설정 : &start=100 (초)
      관련동영상 안보이게 설정 : &rel=0
      다른영상재생 : &playlist=영상아이디       
      반복재생 : &loop=1 (한개 영상으로 계속반복할려면 &playlist 항목에 같은 영상 아이를 지정한다.) 
      상단정보 안보이게 설정  : &showinfo=0 (2018.9 지원종료 ) 
      하단 컨트롤바 안보이게 설정 : &controls=0 
      마우스오버시 컨트롤바 보이게 설정 : &autohide=1  

6. 동영상 반응형으로 수정 

CSS스타일 
1) 동영상 요소의 가로 크기 , 세로크기 모두 100% 지정 
2) 동영상 요소를 감싸고 있는 부모요소의 가로크기 100%, 세로크기는 생략 
( 세로크기를 100% 로 지정하면 동영상 파일의 비율때문에 가로크기가 맞지않음 - 영상화면 비율은 변하지 않는다. )
3) 부모요소에 padding-bottom 값을 동영상 원래크기 비율만큼 %로 지정 
( 부모요소의 높이값을 생략하면 동영상파일은 이미지와 다르게 높이값이 지정되지 않아 동영상이 안보인다 
따라서 padding-bottom 값을 동영상 원래크기 비율만큼 %로 지정하여 높이값을 생성해주고 반응형으로 동영상 높이값 %로 제어하는 편법사용 )

스크립트 
4) PC화면 해상도에 따라 동영상 원래비율과 안맞을 수 있으므로 전체 부모요소를 감싸서 높이값을 스크립트에서 
윈도우 높이값에 맞도록 지정한후 overflow:hidden 처리하여 비율이 안맞는 부분을 안보이게 처리한다.
